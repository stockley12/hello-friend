# La'Couronne - Premium Hair Salon

A modern, mobile-first booking and management system for La'Couronne hair salon.

## Features

- **Client Booking** - Beautiful booking flow with date/time selection
- **WhatsApp Integration** - Instant confirmation via WhatsApp
- **Admin Dashboard** - Complete business management
- **Income Tracking** - Track daily, weekly, monthly earnings in Turkish Lira (₺)
- **Gallery Management** - Upload and manage photos for men & women
- **Availability Settings** - Set working hours and block dates
- **Client Management** - Track VIP clients and booking history
- **Push Notifications** - Get alerted for new bookings with sound

## Tech Stack

- **React 18** with TypeScript
- **Vite 5** for fast development
- **Tailwind CSS 3** with custom dark luxury theme
- **Framer Motion** for animations
- **React Router v6** for navigation
- **TanStack Query** for data fetching
- **shadcn/ui** components

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Admin Access

Default PIN: `1234` (change in Settings)

Admin URL: `/admin/login`

## Mobile Optimization

This app is fully optimized for mobile devices with:
- Touch-friendly UI elements
- Haptic feedback
- Pull-to-refresh
- PWA support for installation
- Safe area support for notched phones
